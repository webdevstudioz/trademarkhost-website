# TrademarkHost Website

🌐 **Live Site:** https://trademarkhost.com

## Overview
TrademarkHost is a modern, conversion-focused website designed to showcase trademark registration and brand protection services. The project focuses on clean design, clear messaging, and a smooth user experience to build trust and generate leads.

## Features
- Clean and professional UI/UX
- Service-focused landing pages
- Mobile responsive design
- Fast loading and optimized structure

## Services Highlighted
- Trademark Registration  
- Brand Protection  
- Legal Consultation  

## Tech Stack
- Webflow (or update if different)
- HTML, CSS, JavaScript
